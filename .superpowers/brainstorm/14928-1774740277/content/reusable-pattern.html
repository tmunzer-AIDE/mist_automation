Key Design Decisions

  • Trigger chip uses the same visual language as the page's existing chips (rounded pill, right-aligned)
  • Expanded section has a subtle blue-tinted background + border to distinguish AI content from data
  • Max height on the analysis section (e.g. 40vh) with internal scroll prevents it from dominating
  • No topbar button needed — the trigger is in the content where it belongs
  • Smooth expand animation via CSS grid-template-rows (0fr → 1fr) for clean height transition
  • Follow-up input stays compact, part of the analysis card
